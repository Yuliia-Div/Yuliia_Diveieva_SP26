1 Task: What operations do the following functions perform: film_in_stock, film_not_in_stock, inventory_in_stock, get_customer_balance, inventory_held_by_customer, rewards_report, last_day?
- film_in_stock Returns a set of available inventory IDs for the film and the store that were chosen.
- film_not_in_stock It does almost the same as the previous function but it returns a list of unavailable inventory IDs
- inventory_in_stock Check whether the film copy is in the store or not by checking if there is a return date
- get_customer_balance Counts the difference between the money that needsto be paid (price of rentals + overdue fines) and what has already been paid
- inventory_held_by_customer The function takes a customer ID and returns a list of not returned films (inventory IDs)
- rewards_report Returns a list of customers based on whether they fit the defined requirements for a reward
- last_day The function returns the last day of the chosen month
